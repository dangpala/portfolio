Computational Problem Solving Infographic

Description

For this assignment, I created an infographic to visually explain the key computational problem-solving concepts: decomposition, pattern recognition, algorithms, and abstraction. The infographic illustrates how a large problem can be broken into manageable parts, how patterns in data help identify solutions, how algorithms provide step-by-step procedures, and how abstraction simplifies complex problems into understandable terms. This project allowed us to creatively communicate these concepts while applying design principles for clarity and visual appeal.
